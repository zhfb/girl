========================================
          💖 AI女友 使用说明
========================================

【快速开始】

1. 双击运行 "AI女友_安装程序.exe"
2. 按照安装向导完成安装
3. 从桌面或开始菜单启动程序
4. 点击 "⚙️ 设置" 按钮
5. 在 "模型设置" 标签页中输入你的 API Key
   - DeepSeek API Key（推荐）
   - 或 OpenAI API Key
6. 点击 "保存设置"
7. 返回聊天界面开始使用！

【功能介绍】

💬 聊天功能
- 实时对话聊天
- 流式输出，打字机效果
- 对话历史自动保存
- 短期和长期记忆

⚙️ 设置功能
- 模型设置：配置 API Key 和模型参数
- 角色设定：自定义女友名字、性格、背景
- 语音设置：选择语音音色
- 记忆设置：调整记忆参数

【获取 API Key】

DeepSeek（推荐，性价比高）
- 访问：https://platform.deepseek.com/
- 注册账号后获取 API Key

OpenAI
- 访问：https://platform.openai.com/
- 注册账号后获取 API Key

【注意事项】

1. 请妥善保管你的 API Key
2. 对话数据保存在本地，不会上传到服务器
3. 用户数据目录：%APPDATA%\AIGirlfriend\
4. 建议定期备份用户数据

【技术支持】

如有问题，请查看项目文档或提交 Issue。

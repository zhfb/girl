========================================
          💖 AI女友 使用说明
========================================

【版本说明】

本项目提供两个版本：

1. 📦 便携版（推荐）
   - 解压后直接运行 AIGirlfriend.exe
   - 数据存储在程序目录下的 data/ 文件夹
   - 可以放在U盘随身携带

2. 🛠️ 安装版
   - 使用安装程序安装到系统
   - 数据存储在 %APPDATA%\AIGirlfriend\
   - 创建桌面和开始菜单快捷方式

========================================
          💖 便携版使用说明
========================================

【快速开始】

1. 下载 "AI女友_v1.0.0_便携版.zip"
2. 解压到任意位置
3. 双击运行 "AIGirlfriend.exe"
4. 点击 "⚙️ 设置" 按钮
5. 在 "模型设置" 标签页中输入你的 API Key
   - DeepSeek API Key（推荐）
   - 或 OpenAI API Key
6. 点击 "保存设置"
7. 返回聊天界面开始使用！

【数据存储】

所有数据都存储在程序目录下的 data/ 文件夹中：
- data/config.json - 用户配置
- data/chat_history/ - 对话历史
- data/memories/ - 长期记忆
- data/voices/ - 语音文件

你可以备份整个 data/ 文件夹来保存你的数据！

========================================
          💖 安装版使用说明
========================================

【快速开始】

1. 下载 "AI女友_v1.0.0_安装版.zip"
2. 解压后运行 "AI女友_安装程序.exe"
3. 按照安装向导完成安装
4. 从桌面或开始菜单启动程序
5. 点击 "⚙️ 设置" 按钮
6. 在 "模型设置" 标签页中输入你的 API Key
   - DeepSeek API Key（推荐）
   - 或 OpenAI API Key
7. 点击 "保存设置"
8. 返回聊天界面开始使用！

【数据存储】

用户数据存储在：%APPDATA%\AIGirlfriend\

========================================
          💖 功能介绍
========================================

💬 聊天功能
- 实时对话聊天
- 流式输出，打字机效果
- 对话历史自动保存
- 短期和长期记忆

⚙️ 设置功能
- 模型设置：配置 API Key 和模型参数
- 角色设定：自定义女友名字、性格、背景
- 语音设置：选择语音音色
- 记忆设置：调整记忆参数

========================================
          💖 获取 API Key
========================================

DeepSeek（推荐，性价比高）
- 访问：https://platform.deepseek.com/
- 注册账号后获取 API Key

OpenAI
- 访问：https://platform.openai.com/
- 注册账号后获取 API Key

========================================
          💖 注意事项
========================================

1. 请妥善保管你的 API Key
2. 对话数据保存在本地，不会上传到服务器
3. 便携版建议定期备份 data/ 文件夹
4. 如遇问题，请查看项目文档或提交 Issue
